A real-time mapping of geolocation's in San Francisco where there is a price surge for Uber Cabs

Rise in Price is a Data Engineering Project for the Galvanize MS in Data Science program.The above map displays a real time stream of geolocation data where price surge is happening for Uber Cabs in San Francisco. There is a job scheduler which runs every min to get real time data using uber API.Kafka is used for data ingestion and stream processing framework Spark Streaming is used to process and transform data .Data is stored in Hbase and UI which is built on Flask framework is used to show the real time location mapping of price surge and doing Batch Analysis.
